Zastępstwa API

REST API systemu zastępstw szkolnych  ·  v1.0.0  ·  JSON

Autoryzacja

Każde żądanie (poza GET /api/) wymaga klucza API:

Nagłówek:  X-API-Key: asiakatalizator

lub parametr:  ?api_key=asiakatalizator

Błędny lub brakujący klucz zwróci 401 Unauthorized.

Ogólne
GET /api/ Informacje o API ▶

Zwraca opis API i listę wszystkich dostępnych endpointów. Nie wymaga autoryzacji.

Przykład odpowiedzi{ "name": "Zastępstwa API", "version": "1.0.0", "auth": { "type": "api_key", "header": "X-API-Key", "param": "api_key" }, "endpoints": [...] }
GET /api/stats Statystyki systemu ▶

Zwraca liczniki danych w systemie, dzisiejsze zastępstwa i top przyczyny nieobecności.

Przykład odpowiedzi{ "counts": { "teachers": 99, "classes": 48, "classrooms": 64, "lessons": 3654, "substitutions": 92 }, "substitutions": { "today": 5, "earliest_date": "2026-03-02", "latest_date": "2026-03-06", "top_causes": { "Zwolnienie lekarskie": 53, "Erasmus-opieka": 29 } }, "generated_at": "2026-03-09T12:00:00+00:00" }
GET /api/search?q= Szukaj nauczycieli i klas ▶

Przeszukuje nauczycieli i klasy po frazie. Minimalna długość frazy: 2 znaki.

ParametrTypOpis
qstringFraza wyszukiwania (min. 2 znaki) – wymagany
GET /api/search?q=Kowalski&api_key=asiakatalizator{ "query": "Kowalski", "teachers": [ { "id": 3, "name": "Kowalski Oskar", "short_name": "Kowalski O" } ], "classes": [], "total": 1 }
Nauczyciele
GET /api/teachers Lista wszystkich nauczycieli ▶

Zwraca posortowaną listę nauczycieli. Obsługuje wyszukiwanie i paginację.

ParametrTypOpis
qstringFiltruj po nazwisku lub skrócie
limitintLiczba wyników (0 = wszystkie)
offsetintPomiń pierwsze N wyników
Przykład odpowiedzi{ "total": 99, "limit": 10, "offset": 0, "count": 10, "data": [ { "id": 1, "name": "Aleksiewicz-Drab Marta", "short_name": "Aleksiewic" }, ... ] }
GET /api/teachers/{id} Szczegóły nauczyciela ▶

Zwraca dane nauczyciela, jego pełny plan tygodniowy i ostatnie zastępstwa (30 dni).

GET /api/teachers/3{ "id": 3, "name": "Kowalski Oskar", "short_name": "Kowalski O", "plan": [ { "day": 1, "day_name": "Poniedziałek", "lessons": [ { "lesson_number": 2, "subject": "Matematyka", "classroom": { "id": 5, "name": "203" }, "class": { "id": 4, "name": "3TGF" } } ] } ], "recent_substitutions": [...] }
Klasy
GET /api/classes Lista wszystkich klas ▶

Posortowana lista klas z pełnymi nazwami. Obsługuje wyszukiwanie i paginację.

ParametrTypOpis
qstringFiltruj po skrócie lub pełnej nazwie klasy
limitintLiczba wyników (0 = wszystkie)
offsetintPomiń pierwsze N wyników

GET /api/classes/{id} Szczegóły klasy ▶

Zwraca dane klasy, pełny plan tygodniowy i ostatnie zastępstwa (30 dni).

GET /api/classes/1{ "id": 1, "name": "3TB", "full_name": "3 technik budownictwa", "plan": [ { "day": 1, "day_name": "Poniedziałek", "lessons": [...] } ], "plan_flat": [...], "recent_substitutions": [...] }

Zastępstwa
GET /api/substitutions Zastępstwa z filtrami ▶

Zwraca zastępstwa z opcjonalnymi filtrami. Wyniki posortowane po dacie i numerze lekcji.

ParametrTypOpis
datestringJeden dzień (YYYY-MM-DD) – skrót dla date_from = date_to
date_fromstringZakres od (YYYY-MM-DD)
date_tostringZakres do (YYYY-MM-DD)
class_idintFiltruj po ID klasy
teacher_idintFiltruj po ID nauczyciela (nieobecny lub zastępca)
limitintLiczba wyników (0 = wszystkie)
offsetintPomiń pierwsze N wyników
Przykład – GET /api/substitutions?date=2026-03-02&api_key=asiakatalizator{ "total": 15, "limit": 15, "offset": 0, "count": 15, "filters": { "date": "2026-03-02", ... }, "data": [ { "id": 1, "date": "2026-03-02", "lesson_number": 1, "subject": "Język niemiecki", "cause": "Erasmus-opieka", "note": "Uczniowie przychodzą później", "has_substitute": false, "original_teacher": { "id": 1, "name": "Aleksiewicz-Drab Marta" }, "substitute_teacher": null, "class": { "id": 1, "name": "3TB", "full_name": "3 technik budownictwa" }, "classroom": { "id": 1, "name": "101" } } ] }
GET /api/substitutions/today Dzisiejsze zastępstwa ▶

Skrót – zwraca zastępstwa na dzisiaj, posortowane po numerze lekcji.

Przykład odpowiedzi{ "date": "2026-03-09", "count": 3, "data": [...] }
GET /api/substitutions/upcoming Nadchodzące zastępstwa ▶

Zastępstwa na najbliższe N dni, pogrupowane po datach. Domyślnie 7 dni, maksymalnie 30.

ParametrTypOpis
daysintLiczba dni do przodu (1–30, domyślnie 7)
Przykład odpowiedzi{ "from": "2026-03-09", "to": "2026-03-16", "days": 7, "total_count": 20, "by_date": [ { "date": "2026-03-10", "count": 8, "substitutions": [...] } ] }
Plan lekcji
GET /api/plan/class/{id} Plan tygodniowy klasy ▶

Pełny plan wszystkich dni tygodnia dla danej klasy. Lekcje pogrupowane po dniach.

GET /api/plan/class/1{ "class": { "id": 1, "name": "3TB", "full_name": "3 technik budownictwa" }, "total_lessons": 35, "week_plan": [ { "day": 1, "day_name": "Poniedziałek", "lessons": [ { "lesson_number": 1, "subject": "Matematyka", "teacher": { "id": 3, "name": "Kowalski Oskar" }, "classroom": { "id": 5, "name": "203" } } ] }, { "day": 2, "day_name": "Wtorek", "lessons": [...] }, ... ] }
GET /api/plan/teacher/{id} Plan tygodniowy nauczyciela ▶

Pełny plan wszystkich dni tygodnia dla danego nauczyciela. Struktura analogiczna do planu klasy.

GET /api/plan/teacher/3{ "teacher": { "id": 3, "name": "Kowalski Oskar", "short_name": "Kowalski O" }, "total_lessons": 18, "week_plan": [ { "day": 1, "day_name": "Poniedziałek", "lessons": [...] }, ... ] }
